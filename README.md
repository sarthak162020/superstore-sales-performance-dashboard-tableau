# superstore-sales-performance-dashboard-tableau
Tableau Data Analysis Project – Sales Performance Dashboard using Superstore dataset

## Project Overview
This project analyzes sales performance using Tableau and the Superstore dataset. 
The dashboard provides insights into sales trends, profitability, customer segments, and regional performance.

## Tools Used
- Tableau
- Excel / CSV Dataset
- GitHub

## KPIs
- Total Sales
- Total Profit
- Total Orders

## Dashboard Features
- Sales Trend Over Time
- Sales by Category
- Sales by State
- Sales by Segment
- Category Wise Analysis

## Insights
- Technology category generates highest sales.
- West region contributes the most revenue.
- Furniture category has lower profit margins.
- Office Supplies provide stable profit.
